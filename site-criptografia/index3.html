A blockchain é um sistema que armazena dados por meio de uma cadeia de blocos interligados, acessível publicamente. Cada bloco contém códigos criptografados, próprios e do bloco anterior, que ajudam a identificá-los.³ Esses códigos são chamados de hashes. Uma vez que um bloco é criado, seu hash é gerado e, se alguma propriedade for alterada, o hash muda. Ou seja, esses elementos são úteis para identificar qualquer alteração, pois toda a cadeia conectada a um bloco precisa ser recalculada quando ele sofre alguma mudança.⁴
